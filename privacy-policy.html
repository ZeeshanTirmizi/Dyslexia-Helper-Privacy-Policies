Privacy Policy for Dyslexia Helper

Last updated: January 2026

Dyslexia Helper does not collect, store, sell, or share personal information.

The extension only saves user preferences, such as whether dyslexia-friendly mode is enabled and which visual theme is selected. These preferences are stored using Chrome extension storage so the extension can remember the user's settings.

Dyslexia Helper does not collect browsing history, page content, personal data, passwords, form entries, analytics data, or location data.

Dyslexia Helper does not transmit user data to any external server.

The extension only modifies the visual presentation of webpages locally in the browser to improve readability.
